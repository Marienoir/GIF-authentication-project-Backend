# GIF-authentication-project-Backend
The API for the frontend authentication for both admin and user

### Requirements
- Node
- npm

### Install Dependencies
```bash
npm install
```

### Environment variable
```bash
.env
```

### Run Application
```bash
npm run start
```
OR

```bash
npm run dev
```
